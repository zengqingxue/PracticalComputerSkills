### ip地址或域名查询

查看当前网络下对应的github服务器ip网站： [站长之家](https://ip.tool.chinaz.com/github.com), 进入该网站后,  

1. 输入github网站地址的域名(*github.com)*即可查询  当前服务器的ip地址
2. 输入某个ip地址, 即可查询其ip地址的地理位置信息及运营商信息
